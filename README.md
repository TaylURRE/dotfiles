## Installation

```bash
cd path/to/dotfiles
source bootstrap.sh
``` 

### Homebrew & Homebrew Cask

```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
sh ~/brew.sh
```
